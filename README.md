*Data Warehouse for Music Streaming Startup (Sparkify)*

Project Overview:
Designed and implemented a data warehouse for a music streaming startup, Sparkify, to analyze user activity and song plays. Built an ETL pipeline to extract JSON logs from S3, stage the data in Amazon Redshift, and transform it into dimensional tables for analytics.

Key Features:

Fact and dimension table design for song play analysis.

SQL transformations for clean, query-optimized data.

Automated ETL pipeline ensuring accurate reporting for analytics teams.

Technologies: Python, SQL, Redshift, S3, ETL, Dimensional Modeling

Project Goals: Enable analytics teams to generate insights on user behavior, song popularity, and streaming trends efficiently.
